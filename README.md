## 弗曼学习法

![](https://i.postimg.cc/T1F4wwcz/a2acc64db9478f9ee3a3fb98ac1163c28441ef9c.png)
<b>诺贝尔物理学奖</b>得主弗曼的学习法，是<b>世界公认最快</b>的学习方法，主要有四个步骤：

1. 选择一个你想理解的知识
2. 设想下你想向别人传授这个知识
3. 如果过程中出现问题就重新回顾这个知识
4. 让你的讲解越来越简单易懂





## 编码技巧

<b>递归控制</b>

* 严格定义递归函数作用，包括参数、返回值、side-effect
* 先<b>一般</b>，后<b>特殊</b>
* 每次调用必须<b>缩小</b>问题规模
* 每次问题规模缩小程度必须为<b>1</b>
